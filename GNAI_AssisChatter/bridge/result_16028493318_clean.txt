**Main Issue:** Dota 2 game crashes with TDR when launching with Vulkan API on Gen12 LP platforms (TGL-U, ADL-S, ADL-P, RKL-S, TWL) due to uninitialized VK_KHR_fragment_shading_rate extension usage.

### 1.2 Detailed Problem Description
**Description:** The issue involves Dota 2 game crashing with a Timeout Detection and Recovery (TDR) error when launched using the Vulkan rendering API on Tiger Lake-U and other Gen12 LP platforms. The root cause was identified as the game attempting to use the VK_KHR_fragment_shading_rate Vulkan extension which is not fully supported on Gen12 LP platforms. The game detects that the extension does not support all required features and reports disabling it, but still calls `vkCmdSetFragmentShadingRateKHR()` function, causing an assertion failure in the driver's `GetFragmentShadingRateOffset` function when `m_FragmentShadingRatePrimitiveCombinerCount` and `m_FragmentShadingRateAttachmentCombinerCount` are zero.

The issue was reproducible at 100% rate on Gen12 LP platforms (TGL-U, ADL-S, ADL-P, RKL-S, TWL) but worked correctly on Gen12 HP (DG2) and MTL-H platforms. The issue was not seen on AMD systems. A fix was implemented via AIL (Application Image List) mechanism to handle the uninitialized extension usage. The issue was verified fixed in driver version 101.7079.
