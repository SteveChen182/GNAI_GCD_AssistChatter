# About

This is a testing mechanism to run the SAT (Sighting Assistant Tool) command and give feedback in one go.

"runs/" folder contains the full SAT command result which ran previously.
"data/" folder contains the feedback that was given by the user associated with the ouptut of the HSD ID.

# Give us Feedback

This testing system requires  **Jupyter Notebook**.

To install Jupyter Notebook using pip, run the following command in your terminal:

BASH

`pip install notebook`

## Usage Guide

1.  Navigate to the  `sighting/accuracy_test`  path.
2.  Start Jupyter Notebook with:
    
    BASH
    
    `jupyter notebook`
    
3.  Open and follow the instructions in the  **`feedback.ipynb`**  notebook.
