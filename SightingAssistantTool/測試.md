# SightingAssistantTool 典型測試指令（10 個）

更新日期: 2026-03-12

## 前置條件

- 目前目錄在專案根目錄
- 已啟用 Python 虛擬環境
- 已安裝並可使用 dt CLI（若公司環境尚未安裝，先略過 dt 指令）

## 測試指令清單

### 1. 檢查 dt CLI 是否可用
```powershell
dt --version
```

### 2. 啟用 gnai extension
```powershell
dt extensions enable gnai
```

### 3. 註冊 SightingAssistantTool
```powershell
dt gnai toolkits register .
```

### 4. 驗證 toolkit 定義是否有效
```powershell
dt gnai toolkits validate .
```

### 5. 基本 HSD 分析（主流程）
```powershell
dt gnai ask "Please assist with the HSD id 14025723680"
```

### 6. 指定 assistant 做摘要
```powershell
dt gnai ask "What is the summary of HSD id 15018275324" --assistant sighting_assistant
```

### 7. 啟用 verbose 觀察工具呼叫細節
```powershell
dt gnai ask "Please assist with the HSD id 14025723680" -v
```

### 8. 以分析導向 prompt 測試完整報告
```powershell
dt gnai ask "Analyze HSD 14026871374" --assistant sighting_assistant
```

### 9. 本地 CLI 測試：log_file_analyzer 說明頁
```powershell
python .\src\log_file_analyzer.py --help
```

### 10. 本地 CLI 測試：sherlog_subprocess 無輸入保護路徑
```powershell
python .\src\sherlog_subprocess.py
```

## 建議執行順序

1. 先跑 1 到 4（環境與工具檢查）
2. 再跑 5 到 8（SightingAssistant 主功能）
3. 最後跑 9 到 10（本地 Python CLI smoke test）
